# Календарь на React (Vite + TypeScript + Tailwind)

Современный, доступный (ARIA + клавиатура) компонент-календарь на React.

## Стек
- React 18 + TypeScript
- Vite
- TailwindCSS
- Без внешних date-библиотек — аккуратные утилиты на чистом JS

## Быстрый старт
```bash
npm install
npm run dev
```

Откройте предложенный адрес (обычно `http://localhost:5173`).

## Сборка
```bash
npm run build
npm run preview
```

## Использование
Откройте `src/App.tsx` — там пример использования компонента `<Calendar />`.
Свойства:
- `value?: Date | null` — выбранная дата
- `onChange?: (date: Date) => void` — колбэк при выборе
- `initialMonth?: Date` — стартовый месяц
- `locale?: string` — локаль для форматирования (по умолчанию `ru-RU`)
- `weekStartsOn?: 0 | 1` — первый день недели (0 — воскресенье, 1 — понедельник; по умолчанию 1)

## Дизайн
В проект подключён Tailwind. Вы можете легко адаптировать внешний вид (скругления, цвета, тени, размеры и т.п.).
